# Follow-up Mailer
CLI tool for automating follow-up emails

---

## Install dependencies
```
npm install
```

## Start application (must have compiled first)
```
npm run start
```

## Start application in development
```
npm run dev
```

## Compile
```
npm compile
```

## Build 
```
npm run build
```
## Lint and fix
```
npm run lint
```
